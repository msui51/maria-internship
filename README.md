## Summary
This project started as a static HTML/CSS/JavaScript single-page application and was expanded into an interactive NFT marketplace experience.

## What I built
- An interactive marketplace UI for browsing NFT content
- A more dynamic frontend experience with loading, navigation, and page transitions
- A cleaner user experience for viewing larger sets of items

## My specific contribution
- Converted a static SPA into a more interactive marketplace
- Added animations to improve polish and user feedback
- Implemented skeleton loading states while content loads
- Added pagination to support browsing through multiple items
- Added dynamic routing for navigating between views/pages
- Integrated REST API-based data handling
- Worked within a Git-based team workflow during development
